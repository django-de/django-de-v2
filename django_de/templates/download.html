Woher bekomme ich Django?

Django ist eine Open-Source-Software, die unter der BSD-Lizenz veröffentlicht wird. Es benötigt Python 2.3 oder höher und hängt von keiner anderen Python-Bibliothek ab. Sie können sich Django auf verschiedene Art und Weise besorgen:

1. Möglichkeit: Laden Sie sich die aktuellste Entwicklerversion herunter

Die neuste und beste Django-Version befindet sich in unserem Subversion Repository (unser Versionskontrollsystem). Laden Sie es sich mit folgendem Kommandozeilenbefehl mittels Subversion herunter:

svn co http://code.djangoproject.com/svn/django/trunk/

2. Möglichkeit: Laden Sie sich die letzte offizielle Version herunter

Die letzte offizielle Version ist 0.96.1 Bitte beachten Sie, dass wir eventuell noch Änderungen an der API vornehmen, bevor wir Version 1.0 veröffentlichen. Bitte lesen Sie die Dokumentation zur API-Stabilität, um mehr dazu zu erfahren. Alle Änderungen, die nicht mit einer älteren Version kompatibel sind, werden von uns detailiert dokumentiert, um eine Aktualisierung zu erleichtern.

So bekommen Sie Version 0.96.1:

Laden Sie zuerst Django-0.96.tar.gz. Danach: 


tar xzvf Django-0.96.tar.gz
cd Django-0.96
sudo python setup.py install

Wenn das Heruntergeladen beendet ist

Lesen Sie die Installationsanleitung für eine genaue Beschreibung.

Schauen Sie auf jeden Fall bei der django-users Mailingliste (englisch) vorbei, wo sich viele Django-Anwender und Entwickler tummeln und sich gegenseitig helfen.

Welche Version ist besser?

Wir verbessern Django fast jeden Tag und sind darin geübt, Code stabil zu halten. Daher ist die neuste Entwicklerversion eine sichere und einfache Möglichkeit, an neue Funktionen zu kommen, noch bevor sie in einer offiziellen Version enthalten sind.

Wenn Sie sich für die Entwicklerversion entscheiden, bedenken Sie bitte, dass es ab und Änderungen vorgenommen werden, die mit einer älteren Version unkompatibel sind. Sie sollten deswegen immer ein Auge auf den Entwicklungsverlauf werden.

Sollten Sie auf der Suche nach einer stabilen Produktivumgebung sein und auf die nächste Veröffentlichung warten können, arbeiten Sie am besten mit der letzten offiziellen Version. Detailierte Informationen über die notwendigen Anpassungen während Aktualisierungen finden sich in jeder neuen Version.
