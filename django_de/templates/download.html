Woher bekomme ich Django?

Django ist eine Open-Source-Software, die unter der BSD-Lizenz veröffentlicht wird, benötigt Python 2.3 oder höher und hängt von keiner anderen Python-Bibliothek ab. Du kannst dir Django auf verschiedene Art und Weise besorgen:

1. Möglichkeit: Lade dir die letzte, offizielle Version

Die letzte, offizielle Version ist 1.0. So bekommst du sie:

Zuerst, lade dir das Django-1.0.tar.gz Paket. Dann:

tar xzvf Django-1.0.tar.gz
cd Django-1.0
sudo python setup.py install

{% comment %}

1. Möglichkeit: Lade dir das 1.0 beta2 Release

Django wird derzeit auf das Release 1.0 im September 2008 vorbereitet. Um den Weg dahin zu ebnen und mehr Augen auf den Quellcode schauen zu lassen, haben wir eine Reihe von "alpha" und "beta" Versionen von Django 1.0 veröffentlicht; diese Versionen sind nicht für den Produktiveinsatz gedacht. Weitere Informationen findest du in den 1.0 beta2 release notes.

Wenn du abenteuerlustig genug bist diesen Code zu testen -- nochmal: nicht in einer Produktivumgebung -- kannst du Django 1.0 beta2 downloaden und mittels python setup.py install installieren.

{% endcomment %}

2. Möglichkeit: Lade dir die aktuellste Entwicklerversion herunter

Die neueste und beste Django-Version befindet sich in unserem Subversion Repository (unser Versionskontrollsystem). Lade dir es mit folgendem Kommandozeilenbefehl mittels Subversion herunter:

svn co http://code.djangoproject.com/svn/django/trunk/

Wenn das Heruntergeladen beendet ist

Lies die Installationsanleitung für eine genaue Beschreibung.

Schau auf jeden Fall bei der django-users Mailingliste (englisch) vorbei, auf der sich viele Django-Anwender und Entwickler tummeln und sich gegenseitig helfen.

Welche Version ist besser?

Django wird von fähigen Entwicklern fast jeden Tag weiterentwickelt und läuft sehr stabil. Deswegen ist die Entwicklerversion die beste Möglichkeit, viele neue Funktionen zu benutzen.

An der Entwicklerversion werden manchmal Änderungen vorgenommen, die mit älteren Versionen inkompatibel sind. Du solltest daher immer ein Auge auf den Entwicklungsverlauf werfen.

Wenn du auf der Suche nach einer stabilen Produktivumgebung bist und auf die nächste Veröffentlichung warten kannst, lade dir bitte die letzte offizielle Version herunter. Jeder neuen Version liegt eine genaue Beschreibung für den Aktualisierungsvorgang bei.
